## 2. 2023/03/26 首届深圳盐田山海半程马拉松 ![](https://www.runchina.org.cn/static/resIcon-DikT0229.png =100x)

::::tabs
@tab 基本情况

> 🗓️ 与上次比赛间隔：7 天

::: info 来自`表格`

🛣️ 路线描述： 后半程坡度较大

☁️ 出门天气： 小雨

🎒 补给准备： 依稀记得是4条迪卡侬能量胶+若干盐丸

⚠️ 意外情况： 无
:::

::: tip 个人感受
和第一次背靠背的马拉松，当时是和吴女士一起奔赴赛场（她只跑5公里的健康跑），也算是家门口的赛事了。当时不知道宝安马拉松能不能中，所以也报了这个。
全程雨战！在终点附近看到了我们BGIer的啦啦队在鼓舞打气。由于70%的路线我在之前日常跑步已经探过路了，所以心里还算有底。
PS：这次雨战竟然还PB了！
:::
